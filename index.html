Learning Evasion Strategies via Reinforcement Learning

Junwon Kim

Abstract

This research explores reinforcement learning in Unity ML-Agents, training an agent to survive by evading dynamically spawning threats. The agent learns optimal movement strategies through curriculum learning and adaptive reward shaping, resulting in a robust evasion policy.

Technical Summary

  • Environment: Unity 2022.3.21f1 + ML-Agents 0.30.0
  • Core Mechanic: Evasion-based survival
  • Reward Design: Survival time + HP loss penalty + threat density penalty
  • Curriculum: Skull speed, spawn count, bullet speed scaled over time
  • Key Result: Simplified reward config (no distance penalty) yielded best results
